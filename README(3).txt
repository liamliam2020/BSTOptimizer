Find Optimal Binary Search Tree

John M. Aronis

October 2018



INTRODUCTION


This program implements a top-down (memoized) dynamic-programming
algorithm to find optimal binary search trees.



INSTALLATION


The following files should be present:

     

BinaryTree.java
     
BSTOptimizer.java
     
TestBSTOptimizer.java
     
README.txt



Compile and test with Java 2.1.5 or later:

     

javac TestBSTOptimizer.java
     
java TestBSTOptimizer



USAGE


See the file TestBSTOptimizer.java for an simple example of usage.



PROBLEMS


No known problems.
